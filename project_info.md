# Project Information
## Project Title: Galaxy Defender

---

## 1. Project Name
Galaxy Defender

## 2. Project Type
2D Browser-Based Space Shooter Game

## 3. Domain
Game Development / Frontend Web Development / Software Engineering

## 4. Project Description
Galaxy Defender is a simple 2D browser-based shooting game developed using HTML, CSS, and JavaScript. In this game, the player controls a spaceship positioned at the bottom of the screen and must destroy enemy villain ships descending from the top. The player earns points by destroying enemies and loses lives when enemy ships collide with the spaceship. The game ends when all lives are lost.

## 5. Objective
The main objective of this project is to design and develop an interactive browser game that demonstrates game logic, user input handling, collision detection, score tracking, and modular code structure using frontend technologies.

## 6. Technologies Used
- HTML
- CSS
- JavaScript
- GitHub (for version control and collaboration)

## 7. Main Modules
- Player Module
- Enemy Module
- Bullet Module
- Collision Detection Module
- Score and Lives Module
- Game Control Module

## 8. Key Features
- Player spaceship movement
- Enemy ship spawning
- Bullet shooting system
- Collision detection
- Score display
- Lives system
- Game Over functionality
- Restart game option

## 9. Expected Output
A functional and interactive 2D browser-based shooting game where the player can move a spaceship, shoot enemy ships, track score, manage lives, and restart the game after losing.

## 10. Future Scope
- Add boss level
- Add power-ups
- Add sound effects
- Add animated explosions
- Add multiple levels
- Add high score / leaderboard system

## 11. Conclusion
Galaxy Defender is a beginner-friendly software engineering project that combines frontend development and basic game mechanics. It provides practical understanding of event handling, animation, collision detection, and modular project organization.
