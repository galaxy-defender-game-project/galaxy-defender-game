# Galaxy Defender

## Project Overview
Galaxy Defender is a 2D browser-based space shooter game developed using HTML, CSS, and JavaScript. In this game, the player controls a spaceship and must destroy enemy villain ships while avoiding collisions. The objective is to survive as long as possible, score maximum points, and protect the galaxy from enemy attacks.

## Features
- Player spaceship movement
- Enemy ship spawning
- Bullet shooting system
- Collision detection
- Score tracking
- Lives / Health system
- Game Over screen
- Restart game option

## Technologies Used
- HTML
- CSS
- JavaScript

## Game Objective
The main objective of Galaxy Defender is to control the player spaceship, destroy enemy villain ships, avoid collisions, and achieve the highest possible score before losing all lives.

## How to Play
1. Open the project in a web browser
2. Use the Left Arrow key to move left
3. Use the Right Arrow key to move right
4. Press the Spacebar to shoot bullets
5. Destroy enemy ships to increase your score
6. Avoid enemy collisions to protect your lives
7. The game ends when all lives are lost

## Controls
- Left Arrow → Move spaceship left
- Right Arrow → Move spaceship right
- Spacebar → Shoot bullet

## How to Run the Project
1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in any modern web browser
4. Start playing the game

## Modules Used
- Player Module → Handles player spaceship movement and actions
- Enemy Module → Handles enemy ship spawning and movement
- Bullet Module → Handles bullet shooting and movement
- Collision Module → Detects bullet hits and collisions
- Score Module → Updates score, lives, and game status
- Game Control Module → Controls game loop, start, restart, and end
